Getting Started

Prerequisites

We strive to keep the Maru application up-to-date using the latest technologies. The following are the current prerequisites your server or hosting platform needs to fulfill for the application to install and run without any problems. Please make sure all are available.

  • PHP 8.1+
  • MySQL 5.7+

File Structure

This is a developer's guide to understand how the application is structured.

Once you have purchased Maru, and extracted the contents of the downloaded application then you will be able to find the zip named dist.zip. Inside this zip file is the main application. Once you further extract that, you will find the folder structure as below.

  • Maru
    • app Laravel default folder
    • bootstrap Laravel default folder
    • config Laravel default folder
    • Modules Main application directory
    • public Laravel default folder
    • storage Laravel default folder
    • vendor Laravel default folder
    • .gitattributes
    • .gitignore
    • artisan
    • composer.json
    • composer.lock
    • module_statuses.json
